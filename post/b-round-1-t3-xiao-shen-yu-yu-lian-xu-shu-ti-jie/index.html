第一个答案，其实就是你要说明：

连续 kkk 个整数的乘积会被多少整除？（最大）

下面我们来证明 连续 kkk 个整数的乘积被 k!k!k! 整除。

错证：

每 iii 个连续整数中，必然有一个 iii 的倍数。

所以有 1,2⋯k1,2 \cdots k1,2⋯k 的倍数。

那么显然 k!k!k! 得证。

这个证法的错误在于：

a∣ca|ca∣c 和 b∣cb|cb∣c ，只能说明 lcm⁡(a,b)∣c\operatorname{lcm}(a,b) | clcm(a,b)∣c ，而不能说明 ab∣cab | cab∣c

就是这个方法只能证明：

lcm⁡(1,2⋯k)\operatorname{lcm} (1,2 \cdots k) lcm(1,2⋯k)

证：

∏i=aa+k−1ik!=(a+k−1)!k!⋅(a−1)!=Ca+k−1k\frac{\prod_{i=a}^{a+k-1} i }{k!}= \frac{(a+k-1)!}{k! \cdot (a-1)!} = C_{a+k-1}^k k!∏i=aa+k−1​i​=k!⋅(a−1)!(a+k−1)!​=Ca+k−1k​

显然组合数是一个整数，所以

k!∣∏i=aa+k−1ik! | \prod_{i=a}^{a+k-1} i k!∣i=a∏a+k−1​i

得证。

第三个答案：

将每个数除以 222 ，然后转为连续整数。

即答案为：

2n⋅n!2^n \cdot n! 2n⋅n!

注明：941∣107+7941 | 10^7+7941∣107+7，所以一些第一个和第三个的答案，大都是 000.

#include<bits/stdc++.h>
using namespace std;

typedef long long ll;
const ll MOD=1e7+7;

inline ll read(){char ch=getchar();int f=1;while(ch<'0' || ch>'9') {if(ch=='-') f=-f; ch=getchar();}
	ll x=0;while(ch>='0' && ch<='9') x=(x<<3)+(x<<1)+ch-'0',ch=getchar();return x*f;}

ll n,fac[941];
ll ans1,ans2,ans3;

//ans1 = n! % MOD
//ans2 = ( (n%2==0) ? (n*2) : n ) % MOD
//ans3 = ( 2^n * n! ) % MOD

inline ll power(ll x,ll y) {
	ll ans=1;
	while(y) {
		if(y&1) ans=(ans*x)%MOD;
		x=(x*x)%MOD; y>>=1;
	} return ans;
}

int main(){
	// freopen("20.in.txt","r",stdin);
	// freopen("20.out.txt","w",stdout);
	n=read();
	fac[1]=1;
	for(int i=2;i<=940;i++) fac[i]=(fac[i-1]*i)%MOD;
	ans1 = (n>=941) ? (0) : (fac[n]);
	ans2 = (n%2==0) ? ((n*2)%MOD) : (n%MOD);
	ans3 = (ans1 * power(2,n) ) %MOD;
	printf("%lld %lld %lld\n",ans1,ans2,ans3);	
	return 0;
}
