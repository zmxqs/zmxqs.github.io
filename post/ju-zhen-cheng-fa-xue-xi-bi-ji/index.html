矩阵乘法学习笔记

2020-03-02 数论

矩阵乘法的规则是：

  1. 两个矩阵必须一个是 n×mn \times mn×m 而另一个是 m×km \times km×k 的形式。

  2. 公式：

Ci,j=∑l=1kai,l×bl,rC_{i,j} = \sum_{l=1}^k a_{i,l} \times b_{l,r} Ci,j​=l=1∑k​ai,l​×bl,r​

  1. 代码：
struct martix{
    int a[101][101];
};
int n;

inline martix chengfa(martix x,martix y) {
	martix ans;
	for(int i=1;i<=n;i++) for(int j=1;j<=n;j++) ans.a[i][j]=0;
	for(int i=1;i<=n;i++) for(int j=1;j<=n;j++) for(int k=1;k<=n;k++) {
		ans.a[i][j]+=x.a[i][k]*y.a[k][j];
		ans.a[i][j]%=MOD;
	} return ans;
}
